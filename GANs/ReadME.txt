This folder consists of the work done using GANs.

The file iitdelhi-biometric.ipynb has the code for GANs that are trained on left-eye images of the IITD-dataset. The dataset on which it has been trained is in the Folder. You can also find the kaggle notebook of this work at https://www.kaggle.com/code/sumanth2002iiitb/iitdelhi-biometric.
