Instructions to run:

1) Install podman
2) Install podman-docker
3) Install podmon-compose as mentioned in https://docs.oracle.com/en/learn/podman-compose/index.html
4) Run the following command:
	sudo ./run.sh --input [DATA_DIR] --mode iris
5) CSV file will be generated in the output folder, copy paste the name into the Skim python notebook
6) pip install numpy matplotlib pandas ipykernel
7) Select output directory in final cell of python notebook
7) Run the notebook
